RiseupVPN is an application to provide easy and secure encrypted communication. The app is a custom branded build of f-droid.org/packages/se.leap.bitmaskclient. This version is preconfigured with one trusted provider—the Riseup collective—and allows anonymous VPN usage without any manual setup.

By encrypting and securely routing all your network traffic through Riseup’s servers, RiseupVPN is able to prevent many forms of censorship and surveillance.
